<h2>210302</h2>

<p>Git</p>
<ul>
<li>gitflog : 커밋을 되돌리고 싶을때 사용함</li>
  <br>
  <pre>
    <code>$ git reflog //커밋 로그 모두 출력</code></pre>
  <br>
  <li>git reflog란 참조의 기록을 보여주는 명령어</li>
  <br>
   <pre>
    <code>git reset --hard HEAD~ //커밋 전으로 돌아가기</code> </pre>
   <pre>
    <code>git log-g </code> / reflog와 커밋을 같이 볼 수 있다.</pre>
</ul>
